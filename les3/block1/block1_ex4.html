Теги для работы с таблицами
Номер Название Описание
1 table Cлужит контейнером для элементов, определяющих содержимое таблицы.
2 tr Cлужит контейнером для создания строки таблицы.
3 caption Предназначен для создания заголовка к таблице
4 col Задает ширину и другие характеристики одной или нескольких колонок таблицы.
5 colgroup Предназначен для задания ширины и стиля одной или нескольких колонок таблицы.
6 tbody Предназначен для хранения одной или нескольких строк таблицы.
7 td Предназначен для создания одной ячейки таблицы.
8 tfoot Предназначен для хранения одной или нескольких строк, которые представлены внизу таблицы.
9 th Предназначен для создания одной ячейки таблицы, которая обозначается как заголовочная.
10 thead Предназначен для хранения одной или нескольких строк, которые представлены вверху таблицы.
